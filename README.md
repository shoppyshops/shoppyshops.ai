# Shoppy Shops

Gen AI assisted management of Shopify, FB and related systems



## Setup

```
poetry shell
poetry install
```
## Environment

```
touch .env
```

Ask for the .env keys or set up your own.

## Usage

We're using [Reloady](https://reloady.dev/) to run the script at the moment during early development.

```
reloady run.py
```
